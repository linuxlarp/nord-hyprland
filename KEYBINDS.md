# Keybinds

These can all be changed in `~/.config/hypr/hyprland.conf`, Please refer to the ![Hyprland documentation](https://wiki.hypr.land/) before making any changes.

---

### Apps & system
- SUPER + T → Open terminal ($terminal)
- SUPER + E → Open file manager ($fileManager)
- SUPER + SPACE → Application launcher ($menu)
- SUPER + Q → Close active window
- SUPER + F12 → Exit Hyprland (shutdown / exit)
- SUPER + L → Lock screen (hyprlock)
- ALT + TAB → Toggle notification center
- SUPER + V → Clipboard menu (cliphist + rofi)

### Layout & navigation
- SUPER + Z → Toggle floating
- SUPER + P → Toggle pseudo-tiling (dwindle)
- SUPER + J → Toggle split orientation (dwindle)
- SUPER + ←/→/↑/↓ → Move focus left/right/up/down
- SUPER + 1–0 → Switch to workspace 1–10
- SUPER + SHIFT + 1–0 → Move active window to workspace 1–10
- SUPER + S → Toggle special workspace "magic"
- SUPER + SHIFT + S → Move window to special workspace "magic"
- SUPER + Scroll Up → Previous workspace
- SUPER + Scroll Down → Next workspace
- SUPER + LMB drag → Move window
- SUPER + RMB drag → Resize window

### Screenshots
- SUPER + Print → Screenshot active window (hyprshot -m window)
- SHIFT + Print → Screenshot current output (hyprshot -m output)
- Print → Screenshot region (hyprshot -m region)
